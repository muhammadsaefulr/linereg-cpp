# Linear Regression With C++

## In This Mini Repo I Have Learned With

- Fit Model Y = a + bX (1D)
- Metrics: MSE, MAE, R²
- Prediction X Value

## The Basic Formula

b = Σ((Xi - X̄)(Yi - Ȳ)) / Σ((Xi - X̄)^2)
a = Ȳ - b X̄
